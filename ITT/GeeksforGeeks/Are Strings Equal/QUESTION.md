Given two strings s1 and s2, find if they contain the same data or not.

Example 1:

Input:
hello hello

Output:
true

Explanation:
Both the strings have same data hello so true is printed.
Example 2:

Input:
geeks skeeg

Output:
false

Explanation:
Both the strings are different so print false.
User Task:
You need to complete the function areEqual that takes s1 and s2 as arguments and returns true if s1 and s2 contain the same data, else it returns false. The driver code then prints it.

Constraints:
1 <= |s1|,|s2| <= 103
